# Models

Versioned writeups of each MaSt model generation.  Each file
documents one model's assumptions, outcomes, limitations, and
the reasons it was superseded.  Root-level files
([`README.md`](../README.md), [`STATUS.md`](../STATUS.md)) point
here for detailed results; [`model-template.md`](model-template.md)
provides a blank structure for new writeups.

- [`model-A.md`](model-A.md) — WvM / single-sheet (S1–S3, R1–R25)
- [`model-B.md`](model-B.md) — Three tori / `ma.py` (R26–R38)
- [`model-C.md`](model-C.md) — Generalized model / `ma_model.py` (R39–R44)
- [`model-D.md`](model-D.md) — **Active** — Filtered model / `ma_model_c.py` (R45–R50+)
