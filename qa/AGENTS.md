# Questions and Answers

- When the user wants to capture questions for later processing, enter them into the INBOX
- To process a question, create a Q file that restates the question and expands by answering, explaining, articulating
- When answering, make it clear when the answer is an appeal to standard physics or the MaSt/GRID framework
- Number Q files sequentially
- Keep a current, organized index of Q files in qa/README.md
