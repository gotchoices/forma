# Q50. Shared material space

**Status:** Open — conceptual/foundational
**Source:** claude-photon-knots-particle-structure.md
**Model era:** Model A

Is the material T³ a single space shared by all particles — each particle a distinct topological feature of one global manifold — or does each particle carry its own private subspace? A shared T³ makes entanglement and interactions geometric; independent subspaces push the mystery down a level. The dialog notes that all mature frameworks (AdS/CFT, string theory, LQG) favor a single shared space. What does the WvM/T³ model imply?
