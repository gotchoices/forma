# Q&A

Questions answerable by appeal to physics knowledge, established theory,
or logical argument — no computational study required.

For questions that need a computational model, see [`studies/`](../studies/).

## Workflow

1. **Capture** — New question → add to [`INBOX.md`](INBOX.md).
2. **Triage** — Is it answerable from first principles / existing physics?
   - Yes, and brief → answer inline in INBOX, mark `answered`.
   - Yes, but substantial → write it up as `Q<N>-slug.md` here.
   - No, needs computation → create a study entry in [`studies/STATUS.md`](../studies/STATUS.md).
3. **File** — Substantial questions (answered or open-but-rich) get their
   own `Q<N>-slug.md` file. Add a one-line entry to the index below and
   remove the INBOX entry.

[`INBOX.md`](INBOX.md) is strictly a capture queue for untriaged questions.
Once a question is answered or promoted to a Q file, it leaves the INBOX.
This README is the sole index of all Q files.

**Note:** Q105 has two files (duplicate numbering) — listed as Q105a and
Q105b pending renumbering.
