# Theory of the Universe

This folder is a workspace to explore ideas about the physics of atoms and sub-atomic particles.

Of particular interest is a paper by Williamson and van der Mark ([PDF](https://fondationlouisdebroglie.org/AFLB-222/MARK.TEX2.pdf)) in which they propose the possibility that an electron is composed of a photon bound in a toroidal orbit.

The intent is to make a workbench containing one or more study folders, each exploring a single thread of reasoning.

## Structure

- `ref/` — Reference material (papers, source dialogs).
- `studies/` — Individual studies, each in its own folder.
