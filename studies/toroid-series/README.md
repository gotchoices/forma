# Toroid Series

This study focuses on a possible refinement of the Williamson–van der Mark (WvM) paper.

## Observations

- We adopt the toroidal confinement idea from WvM.
- The WvM model predicts an electron charge of ~1.46×10⁻¹⁹ C (≈ 0.91*e*, Eq. 5 in the paper), which falls short of the measured value of 1.602×10⁻¹⁹ C.
- WvM also fails to identify any force strong enough to confine a photon to such a tight orbit.
- String theory addresses many problems in subatomic physics but requires extra dimensions (10 or 11), whereas we typically observe only 4 at the macro level.
- String theory has posited that extra dimensions might be microscopic structures, possibly fractal in nature.

## Questions

- Is it possible that the charge calculated by WvM is just the first term of a series, and so only approximates the full charge?
  - Does the ratio of the first series element to the final value correspond to any recognizable constant (such as the golden ratio or the fine-structure constant)?
- Is it possible that the toroid is a sub-dimension rather than a confinement in our 3D space?
  - If so, it might not be necessary to "constrain" the orbit, as the photon would be following a straight line from its own perspective (a geodesic).
- Is it possible that there are fractal dimensions below the initial toroid, such that the photon traverses a course through multiple nested dimensions?
  - Might these dimensions be spaced according to natural harmonics in some way?

3. See if a series can be written from this model.
4. Test the series by writing a program to compute it.
5. Determine if it converges to the measured electron charge.
6. Experiment with different configurations and spacing of extra sub-dimensions.
