# R3. Dual Visualizer  *(concluded)*

Interactive side-by-side animation showing a photon on a (1,2) geodesic:
a 3D torus on the left, the equivalent flat 2D rectangle on the right,
with synchronized motion. Demonstrates that geodesics which look curved
on the embedded torus are straight lines on the flat T².

**Deliverable:** [`index.html`](index.html) — self-contained browser app.

**Key point:** The flat-rectangle view is the physically correct picture.
The winding number (1,2) is immediately legible as a slope on the flat
rectangle, and the photon's straight-line path confirms the flat-T²
framework adopted in R8 and beyond.

**Status:** Concluded. No findings file — the visualization is the result.
